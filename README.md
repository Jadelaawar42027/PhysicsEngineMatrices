# PhysicsEngineMatrices

# 3D Physics & Light Visualization Engine

A real-time 3D rendering engine built with **Python**, `pygame`, and `numpy` — no 3D libraries. Demonstrates core Linear Algebra concepts: transformation matrices, perspective projection, dot/cross products, vector reflection, and shadow casting onto a ground plane.

## Installation

```bash
git clone https://github.com/Jadelaawar42027/PhysicsEngineMatrices
cd your-repo
pip install pygame numpy
python engine.py
```

## Controls

Use the right-hand panel sliders to move the object, reposition the sun, and adjust shadow intensity. Toggle between **Cube**, **Pyramid**, and **Prism** with the shape buttons. Press **R** to toggle auto-rotation, **ESC** to quit.
